Я сделал свой клавиатурный тренажер.
На выбор в тренажере будет даваться 3 уровня сложности: Easy, Medium and Hard
Эти три теста находятся в трех разных .txt файлах, которые также прикреплены.
Все тесты можно менять, достаточно изменить текст в одном из файлов.
Подсчитывается количество ошибок и время прохождения.
Также поддерживается история попыток.
Для запуска достаточно запустить .py файл:
      python3 ./game.py
